In order to add the Compare My Move Affiliate Form to your WordPress site, you need to download our plugin. Everything else is ready to go.

== Description ==
This plugin allows you to put our Affiliate Quotation Form on your WordPress site with ease. Before the plugin can be used you will need to sign up to our affiliate program to get your affiliate code.

Compare My Move is a Removal comparison site based in the United Kingdom. We compare removal companies all over the UK for our visitors to find the best price company to suit their needs.

If you are interested in becoming an affiliate, please visit: http://www.comparemymove.com/affiliates/

== Installation ==
1. Before you install this, you should consider setting up an affiliate account with Compare My Move http://www.comparemymove.com/affiliates/.
2. Once you have been designated an Affiliate code you can install this plugin. Just go to Plugins and either select Add New or search for it in the WordPress Plugin directory.
3. Once you have done this, click Activate.
4. Finally visit the link on the left menu bar and add your Affiliate code.
5. Click save changes and then follow the instructions on the page.
